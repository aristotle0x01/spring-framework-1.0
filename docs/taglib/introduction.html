Spring Framework Tag Documentation

One of the view technologies you can use with the Spring Framework is Java Server Pages. To help you implement views using Java Server Pages the Spring Framework provides you with some tags for evaluating errors, setting themes and outputting internationalized messages.

The Spring Framework does not aim to provide a full-fledged set of tags, useful for all kinds of purposes. Teh tags the Spring Framework provides are just tags to make the Spring Framework work with Java Server Pages technology.

For more information on the general concepts of the MVC framework Spring comes with, refer to the MVC Step by Step tutorial or the Reference Document - Chapter 11. Web framework.
